Exercise 0 - document.getDocumentById()

This exercise requires you to write a small javascript function that will modify the text on level02/exercise00.html

  1. Locate the HTML element you want to change and note its id
  2. In the javascript function, define a variable saying to hold the element
  3. Retrieve the DOM element document.getElementById(...) and assign it to saying
  4. Change the value of the element by using saying.innerHTML = ..., choose any text you like
Hint: don't forget to register an event to trigger the function!

Exercise 1 - document.createElement()

In this exercise, you will write javascript code that dynamically creates new nodes in the DOM. Download and open the HTML file level02/exercise01.html in your browser. You will see three colored squares. You must write code that generates one hundred additional squares of varying colors.

  1. Create new div elements with document.createElement("div")
  2. Add and set the class attribute with square.className="..."
  3. Access the CSS background-color property using square.style.backgroundColor
  4. Now repeat the above steps 100 times and assign different colors each time.

Exercise 2 - Message Box

Have a look at level02/exercise02.html. The goal of this exercise is to write javascript code that will print user messages on the page. The newest messages should appear on top, above older messages. Fill out the content of the function print_message():

  1. Access the node of the input element where the user entered the message. (hint: use document.getElementById(...))
  2. Read the message with the .value field. E.g. if you called the node variable input_elem, the message will be found in input_elem.value
  3. Create a new div element with document.createElement("div")
  4. Set the content of the div according to the user message, use the .innerHTML for this.
  5. Find the div with id message_box
  6. If there are no children nodes yet, append the newly created div using .appendChild(...).
  7. If there already are children, use .insertBefore(<newNode>, <firstChild>)
Good luck!
