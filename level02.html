Exercise 0 - getDocumentById()

This exercise requires you to write a small javascript function that will modify the text on level02/exercise00.html

  1. Locate the HTML element you want to change and note its id
  2. In the javascript function, define a variable saying to hold the element
  3. Retrieve the DOM element document.getElementById(...) and assign it to saying
  4. Change the value of the element by using saying.innerHTML = ..., choose any text you like
Hint: don't forget to register an event to trigger the function!
