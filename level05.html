Exercise 0 - AJAX Form Validation

In this exercise, you will implement an HTML form validation using AJAX. The backend already has all the validation logic. Your task is to call the PHP file, check if there are validation errors, and if so, display the corresponding validation messages in the form beside the corresponding field. Download level05/exercise00.html and level05/exercise00.php into your web server root.
